# Генерация YML-фида каталога для поисковых систем
(автоматическая генерация)

Этот модуль предоставляет возможность генерации YML-фида (Yandex Market Language) для каталога товаров, который можно использовать для интеграции с поисковыми системами и маркетплейсами.

## Использование

### Генерация YML-фида

Для генерации YML-фида выполните следующую команду:

```bash
python manage.py generate_yml_feed
```

По умолчанию файл будет сохранен в директории `static/catalog_feed.yml`.

### Указание пользовательского имени файла

Вы можете указать пользовательское имя файла с помощью параметра `--output`:

```bash
python manage.py generate_yml_feed --output=my_custom_feed.yml
```

## Доступ к YML-фиду

После генерации YML-фид будет доступен по URL:

```
https://angel-dress.ru/catalog/yml-feed/
```

## Интеграция с поисковыми системами

### Яндекс.Маркет

Для интеграции с Яндекс.Маркетом используйте следующий URL:

```
https://angel-dress.ru/catalog/yml-feed/
```

В файле `robots.txt` уже добавлена директива `Yandex-Market` для автоматического обнаружения фида.

### Google Merchant Center

Для интеграции с Google Merchant Center используйте тот же URL:

```
https://angel-dress.ru/catalog/yml-feed/
```

## Структура YML-фида

YML-фид содержит следующую информацию о товарах:

- Название товара
- Описание
- Цена аренды
- Залог (если указан)
- Категории
- Изображения (до 10 штук)
- Характеристики товара (размеры, цвета, материалы и т.д.)
- Срок аренды

## Автоматическое обновление

Рекомендуется настроить регулярное обновление YML-фида с помощью cron-задачи. Например, для обновления фида каждый день в 3 часа ночи добавьте следующую запись в crontab:

```
0 3 * * * cd /path/to/your/project && python manage.py generate_yml_feed
```

## Примечания

- В YML-фид включаются только активные товары (`is_active=True`).
- Товары без цены не включаются в фид.
- Для каждого товара используется минимальная цена из активных вариантов цен.
